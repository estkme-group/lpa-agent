# LPA Agent

## Quick Start
Run the program

```bash
./lpa-agent --help
Usage of lpac-agent:
  -download
        download lpac (default true)
  -listen string
        listen address (default ":9527")
  -lpac-dir string
        lpac directory (default "/home/user/workspace/lpa-agent/lpac-cli")
  -reader string
        pcsc or at (default "pcsc")
  -readerName string
        reader name
```
